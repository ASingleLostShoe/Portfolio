Building with Claude Code

AI-Assisted Development 2026

Overview

This portfolio website was built almost entirely through conversation with Claude Code, Anthropic's AI coding assistant. The project started as a migration from a Coda.io portfolio document into a standalone static website, and evolved into a fully-featured portfolio with 13 individual project pages, interactive galleries, and embedded Story Maps.

The design takes inspiration from the Field Operations website, adopting a minimal white and gray aesthetic with lowercase typography and clean grid layouts. Every line of HTML, CSS, and JavaScript was generated through iterative prompting — describing what I wanted, reviewing the output, and refining through follow-up conversations.

Technical Details

  • Static HTML/CSS/JS with zero external library dependencies
  • Dynamic tile system using CSS Grid with grid-auto-flow: dense and a custom row-packing algorithm that randomizes tile sizes on each page load
  • Responsive breakpoints scaling from 6 columns on wide screens down to 1 column on mobile
  • 13 project detail pages with lightbox image galleries, PDF preview modals, and ESRI Story Map embeds
  • Hosted on GitHub Pages with custom domain (pathallis.me) and enforced HTTPS
  • Security hardening including rel="noopener noreferrer" on external links, iframe sandboxing on Story Map embeds, and PDF metadata stripping

Challenges & Iterations

The development process wasn't always linear. Several approaches were tried and either refined or abandoned entirely:

  • Dark mode was implemented using prefers-color-scheme but wasn't detecting correctly, so it was reverted to a light-only theme
  • Masonry.js was initially used for the project grid, but CDN-hosted SRI hashes kept breaking due to floating version ranges — the entire library was replaced with a zero-dependency CSS Grid solution
  • Tile sizing evolved from static assignments to a row-packing algorithm that ensures column spans always sum to 6, with randomized tall tiles for visual variety
  • Security concerns required purging CLAUDE.md from git history using git-filter-repo and stripping local file paths from PDF metadata with exiftool
